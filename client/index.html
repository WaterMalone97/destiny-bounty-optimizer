Destiny Bounty Optimizer

Welcome to the Destiny Bounty Optimizer

This application will help you find the most efficent bounites to complete to get the most experience in shortest amount of time in Destiny 2. It will utilize Bungie's API to gather information from all the vendors and send it to the application where it will interpret the data and display the route that the user should take. Users will be able to filter any type of bounties that they wish to complete or ignore.
